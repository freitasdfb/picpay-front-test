# Picpay Frontend Test
Este é um projeto para o processo seletivo do picpay
Foi criado a partir de: [Create React App](https://github.com/facebook/create-react-app).

## Scripts para execução

No diretório do projeto, após instalar suas dependências com 

`yarn install`,

Rode o comando 

`yarn start`

para iniciar o projeto.

Para login na interface, só é necessário informar 
`admin` no campo de email e senha

A aplicação estará disponível em  [http://localhost:3000](http://localhost:3000) 

## Bibliotecas utilizadas

Como biblioteca principal para desenvolvimento: [React](https://reactjs.org/)

Como biblioteca padrão para componentes visuais: [react-bootstrap](https://react-bootstrap.github.io/)

Para estilização e criação de componentes visuais: [styled-components](https://styled-components.com/)
